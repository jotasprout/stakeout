For debugging print styles in Firefox open the Developer Toolbar (Shift + F2 or Tools - Web Developer - Developer Toolbar) and enter media emulate print in the input field at the bottom of the browser window and press enter. The active tab will act as if the media type was print until you close it or refresh the page.

In Chrome open DevTools (CMD + Opt + I (macOS) or Ctrl + Shift + I (Windows) or View - Developer - Developer Tools) and show the console drawer (Esc), open the rendering pane, check Emulate CSS Media and select Print.
